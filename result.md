SpaceX（SPCX）于2026年6月12日以 **$135/股** 在纳斯达克上市，是 **2026年全球最具标志性的IPO**。当前股价 $162（+20%），市值 **$2.13万亿**，是美股第三大市值公司（仅次于苹果、微软）。

## 3️⃣ 业务与财务质量分析

### 3.1 公司卖什么？竞争优势能持续多久？

SpaceX 拥有三个核心业务板块：

#### ① 发射服务（Space Segment）

| 项目 | 内容 |
|:----|:------|
| 主力产品 | Falcon 9（可回收中型火箭）、Falcon Heavy（重型）、Starship（下一代超重型） |
| 核心竞争力 | **全球唯一掌握轨道级火箭可重复使用技术的商业公司** |
| 成本优势 | 发射成本仅为竞争对手（ULA、Arianespace）的1/3至1/5 |
| 市场份额 | 2025年全球轨道发射次数占比超过60%，实际处于垄断地位 |
| 核心客户 | NASA、美国太空军、商业卫星运营商、国际客户 |

#### ② Starlink 卫星互联网（Connectivity Segment）

| 项目 | 内容 |
|:----|:------|
| 在轨卫星数量 | 超过7,000颗（截至2026年中） |
| 活跃用户 | 全球超过500万 |
| 收入增速 | 远快于发射业务，正在成为SpaceX最大收入来源 |
| 竞争优势 | 低延迟+全球覆盖+成本规模优势，短期内无实质性竞争对手 |
| 潜在市场 | 全球约30亿人缺乏可靠宽带接入 |

#### ③ 政府/国防业务（Starshield）

- 为美国及盟国政府提供卫星通信、遥感、安全航天服务
- 高利润率、长周期合同、战略性壁垒极高

#### 竞争优势持续性判断

**可维持5-10年以上。** 火箭可重复使用技术建立了巨大的成本护城河，Starship 若成功将把入轨成本再降一个数量级。Starlink 的轨道资源+制造规模优势，使得后来者极难追赶。唯一的变数是技术突破（如Amazon Kuiper的部署进度）和监管政策变化。

## 4️⃣ 估值分析

### 4.1 估值方法论选择

SpaceX **不适合 PE 估值**（持续亏损），需结合多种方法：

| 方法 | 适用性 | 优先级 |
|:----|:------:|:-----:|
| PS（市销率） | ✅ 适用于高增长尚不盈利公司 | ★★★★★ |
| EV/Sales | ✅ 考虑资本结构差异 | ★★★★ |
| EV/EBITDA | ✅ EBITDA已转正，有参考价值 | ★★★★ |
| PB（市净率） | ⚠️ 重资产行业参考 | ★★★ |
| DCF（现金流折现） | ⚠️ 假设过多，不确定性大 | ★★ |
| PE（市盈率） | ❌ 净利润为负，无法使用 | 不使用 |

### 4.2 市销率（PS）估值

| 指标 | 当前（基于$162, TTM） | FY2026E（一致预期） | FY2027E（一致预期） |
|:----|:-------------------:|:------------------:|:------------------:|
| 收入 | $19.30B | **$36.85B** | **$68.25B** |
| **PS** | **110.6x** | **~47.1x** | **~25.4x** |
| EV/Sales | 110.9x | — | — |

**解读：**
- **PS 110x 在任何行业都是极高估值**，放在整个美股市场也属于顶尖水平
- 但若FY2026收入+97%的一致预期兑现，Forward PS 将降至 ~47x
- 若FY2027再+85%至 $68B，PS 将降至 ~25x
- 若公司最终实现 15-20% 净利率，25x PS ≈ 4-5x PE——这对垄断级航天基础设施公司不算离谱

**结论：当前估值隐含了极其乐观的增长路径。如果收入增速<预期，估值收缩空间巨大。**

### 4.3 EV/EBITDA 估值

| 指标 | 数值 |
|:----|:----:|
| 企业价值 | $2.14万亿 |
| TTM EBITDA | $3.95B |
| **EV/EBITDA** | **542x** |

542x EV/EBITDA 在成熟公司中绝无仅有。但若 Starlink 完成部署后资本开支下降、EBITDA 大幅扩张（例如达到 $150-200亿），倍数将快速下降至 10-15x，进入合理区间。

**核心矛盾：当前价格买的是"未来的EBITDA"，不是"现在的EBITDA"。**

**关键洞察：**

2. 最近似的可比对象是 **TSLA（特斯拉）的巅峰期估值**——明星CEO溢价+颠覆性技术叙事+巨大潜在市场
3. Rocket Lab（RKLB）PS ~20x 是SpaceX的1/5，但RKLB在规模和业务广度上远不及SpaceX

### 4.7 DCF 估值思路（定性分析）

不做具体DCF模型（假设过多），但分析市场价格隐含的预期：

$2.13万亿市值，对应当前 $19.3B 收入。思考情景：
- **情景A（保守）**：10年后收入$100B，净利率15%，PE 20x → 市值 $300B → 当前价格过高
- **情景B（中性）**：10年后收入$300B，净利率18%，PE 22x → 市值 $1.19T → 当前价格略高
- **情景C（乐观）**：10年后收入$500B，净利率20%，PE 21x → 市值 $2.1T → 当前价格合理
- **情景D（极度乐观）**：Starship开辟火星运输、太空制造等全新市场，总收入$1T+ → 当前价格偏低

**结论：当前市值隐含了情景C以上的预期。** Starship的商业化成功、Starlink的全球主导地位缺一不可。
